User Guide

Welcome to the Progress Tracker! This guide explains how to use all the features of the application.

Your Data, Your Privacy

All data entered is saved **exclusively in your browser's local storage**. This means your information stays on your device and is never sent to any external server. If you clear your browser data, switch browsers, or use incognito/private mode, your data will be lost unless you have previously exported it.

Understanding the Interface

The main page is organized into collapsible sections. Here’s what each one does:

This is where you record a new activity or achievement. To add an event:

  1. Date & Time: The current date and time are pre-filled. You can click to change them if you're logging a past event.
  2. Numeric Value: Enter the value for your activity (e.g., `15` for push-ups, `5.2` for kilometers run). Decimals are allowed.
  3. Event Type: Select an existing category from the dropdown menu.
  4. Or new type: If the category doesn't exist, type a name in this field (e.g., "Pages Read") and click the Add Type button. It will be saved for future use.
  5. Notes (Optional): Add any relevant details about the event.
  6. Click the Add Event button to save your record.

This section gives you a visual overview of your individual entries over time.

  • The Chart: Shows each event you've logged as a point on the graph. This is useful for seeing your performance on a specific day and identifying trends.
  • Max Values Panel: To the right of the chart, this panel shows the all-time highest value you have recorded for each event type. It's a quick way to see your personal bests.
  • Filter: Use the dropdown menu above the chart to show data for all activities or just a specific one.

This chart is different from the first one. It calculates the **sum of all values for each day** and displays the total.

Example: If you log 20 push-ups in the morning and 15 in the evening on the same day, the "Activity Trend Chart" will show two separate points (20 and 15), while this "Daily Totals Chart" will show a single point at 35 for that day.

It's perfect for tracking your total daily volume for a given activity.

This section provides a simple table view of your most recent entries, sorted from newest to oldest.

You can delete any event permanently by clicking the trash icon in its row.

Data Management

In the top-right corner of the navigation bar, you'll find three essential data management tools:

  • Export Data: Click this to download a full backup of all your event types and recorded events. The data is saved in a .json file. It's highly recommended to do this periodically!
  • Import Data: This allows you to restore your data from a previously exported .json file. This is useful for moving data to a new device or browser.

Warning!

Importing data will overwrite all current data in the browser. There is no "undo" button. Similarly, clicking Clear All Data will permanently erase everything. Use these features with caution.
