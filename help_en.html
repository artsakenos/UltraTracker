User Guide

Welcome to the Progress Tracker!

Adding an Event

To record a new progress (event):

  1. Date & Time: The current date and time will be automatically entered. You can modify them if necessary.
  2. Event Type: Select an event type from the list (e.g., "Push-ups", "Sit-ups"). Next to the dropdown, there's a trash icon button to delete the currently selected event type (if not in use).
  3. Or create new type: If the event type you want is not present, type it into the "Or create new type" field and click "Add Type". The new type will then be available in the list for future selections.
  4. Numeric Value: Enter the numerical value of your progress (e.g., number of push-ups, km run, weight lifted). You can use decimal numbers.
  5. Notes: Add any details or comments about the event.
  6. Click "Add Event" to save.

Dashboard

  • Latest Recorded Events: A table shows the most recent events you have recorded. You can delete an event by clicking the trash icon.
  • Activity Trend Chart: A graph displays the trend of your progress over time. You can filter the data shown in the chart by selecting a specific event type from the dropdown menu above the chart.

Data Management

In the top-right navigation bar, you will find buttons for:

  • Export Data: Download all your data into a JSON file. Keep this file for backup or to transfer data.
  • Import Data: Upload a JSON file (previously exported from this application) to restore your data. Warning: Importing will overwrite currently present data.
  • Clear All Data: Permanently removes all event data and custom event types from your browser. Use with caution!

Data and Privacy

All data entered into this application is saved exclusively in your browser's local storage. This means the data remains on your computer and is not sent to any external server. If you change browsers, use incognito mode, or clear your browser data, the application data will be lost unless you have exported it.
